Wireguard + Privoxy 轉發解鎖區域限制

於 Astrill VPN 下載 wireguard.conf 設定檔

Memberzone > VPN and IP Testing Tools > Wireguard 3rd party configuration

自訂名稱，選擇 Server 之後，下載 config 檔。下載完成後，可以自行命名 config 檔，例如 wgca.conf

[注意]Astrill VPN為付費的VPN供應商，需要訂閱付費後才能使用。

於 Raspberry Pi 安裝 wireguard

於終端機視窗輸入指令：

sudo apt install wireguard

接著輸入：

sudo wg-quick up wgca

然後輸入：

sudo wg

進行檢查。如果有跑出一堆連線資訊，就代表你連線成功了。

安裝 Privoxy

小弟一直有聽說這個軟體，但是一直想不到應用的場景，直到詢問 Gemini 之後，才發現 Privoxy 很好用。

於終端機視窗輸入指令：

sudo apt update
sudo apt install privoxy -y

編輯設定檔：

sudo nano /etc/privoxy/config

將檔案中的 listen-address 127.0.0.1:8118 改為：

listen-address 0.0.0.0:8118

重新啟動 Privoxy 服務：

sudo systemctl restart privoxy

使用案例

小弟在 Google TV Streamer 中的 設定 > 網際網路 > Proxy 設定

IP 位置輸入你安裝 Privoxy 的樹莓派的 IP 位置 (例如 192.168.X.X)

Port 輸入預設的 8118

設定完成後，如果沒問題，你應該就可以解鎖台灣區以外，例如加拿大區的影音串流內容了。

小弟目前用這招解鎖 Paramount+、HBOMAX、KaroStream 等平台都沒問題。
