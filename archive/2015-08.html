[Panasonic麵包機]Panasonic SD-BH1000T麵包機

我家女王在一次拜訪樓下鄰居的過程中，吃到了由Panasonic麵包機做出的麵包之後，對於Panasonic麵包機做出來的麵包的手感念念不忘。於是接下來就是我們搜尋麵包機的過程。原先我們有在網路上找鄰居的麵包機型號SD-BMT1000T，不過由於價格超出我們的預算(我們有預算嗎？)太多，所以我們就去百貨公司找。結果百貨公司也只有賣高等級的機型，我們就用價格的方式來縮小我們的選擇範圍，最後在價格方面就只有SD-BMS105T、SD-BM103TS、SD-BH1000T符合我們的要求。不過SD-BMS105T價格稍貴、 SD-BM103TS功能又太陽春，因此最後就是選擇SD-BH1000T啦！

此外，SD-BH1000T的粉紅色系也是勝出的關鍵，若不是因為粉紅色，可能還真的很難選擇呢！

在經過多次試驗之後，女王已經抓住做麵包的訣竅了！實際上這台要上手真的很簡單，只要照著MENU上的指示，然後在麵包機上按一按，機器就會自動開始工作，所以幾乎是不會失敗的！而且自從開始用麵包機做麵包之後，每天早上女王都是被麵包的香味叫醒，我也好想要這種天天被麵包(機)叫醒的生活呀 XD

做好後的麵包就像這樣，光看圖就好像聞到香味了！

接著就要準備把麵包拿出來囉！

倒出來的麵包，看起應該是成功的，酵母發酵的很好，沒有塌陷。

我只能說這台麵包機真是好物呀！

尤其是現在食安問題嚴重，加上之前胖達人事件，麵包機真的是我們生活當中不可缺少的工具呢！

[HP Stream 13]WIN8.1 with Bing升級為WIN10

自從微軟開放WIN7和WIN8.1免費升級之後，我就想說我的HP Stream 13筆記型電腦是否也可以升級成WIN10。雖然之前有在網路上爬過文，說HP STREAM 13有可能無法升級(因為硬碟壓縮的關係)，以下式這段敘述的原文。

Earlier releases of Windows 10 allowed for an in-place upgrade of a Windows 8.1 WIMBoot system.  The “Windows can’t be installed because this PC uses a compressed operating system.” error was introduced with the latest Windows 10 build 9926 so hopefully later releases and the final release will not have this restriction. 

不過我在升級的過程中，發現微軟會有將隨身碟插入的選項，這樣似乎就可以解決升級時硬碟的問題了。

如果你已經有順利接獲通知，並且有訊息提示你要更新(升級)，你應該就可以看到下面的畫面。

接著系統就會開始準備升級前的前置作業。

經過很長的一段時間，終於升級完成了，看到這樣的升級成功畫面，有沒有很感動？

WIN10的賣點之一，Edge瀏覽器，感覺上跑起來就比IE輕快許多，而且還有一些我覺得不錯的功能。

Edge結合Onenote的功能，可以把在Edge上做的筆記儲存在Onenote上，這樣跟別人分享真的很方便呢！

另外，Edge瀏覽器另一個很讚的功能就是閱讀模式。以下是網頁原先的排版。

開啟閱讀模式之後，是否覺得清爽許多呢？

雖然升級成WIN10之後，炫麗的動態磚被被縮至開始列，但是這樣的設計應該會使大家更能接受WIN10吧！我用到現在，我一點都不後悔升級成WIN10，況且微軟這樣免費的大手筆，加上Edge瀏覽器這樣強大的功能，這一定要支持一下的啦！
