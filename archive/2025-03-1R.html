Picoreplayer-將音樂轉發至homepod上

Picoreplayer-每次只要我的樹莓派Zero W卸下某個階段性的任務的時候，我就會幫它開始找下一個任務。尤其是這台Zero W效能並不是很高，能夠執行的任務及功能越來越少，加上Zero W又沒有有線網路孔，可以應用的範圍就更少了。就在想不到要如何賦予Zero W任務的時候，就想到之前沒用過picoreplayer這套音樂管理及播放軟體，於是我就安裝了這套軟體。

後來發現picoreplayer這套軟體雖然架構很簡單，但是只要安裝LMS還有其他插件，就可以變身成一套功能強大的音樂播放軟體。加上還有支援藍芽播放，以及支援將外界input音樂轉發至其他renderer或裝置的功能，實在不得不叫人躍躍欲試啊！由於我剛好有黑膠音樂轉發至homepod的需求，所以我就索性把這台picoreplayer拿來，把黑膠音樂轉發到homepod上。

所需的材料有：樹莓派一台、USB音效擷取卡、一台支援airplay播放協定的音響、一張32G的micro SD卡、picoreplayer的iso檔

首先要把從網路上抓下來的picoreplayer的iso檔，用etcher軟體燒入micro SD卡上。若是你的樹莓有有線網路孔，你就可以直接插網路孔就可以連上網路，否則就要在開機前先在micro SD的設定檔上，設定wifi帳號密碼（詳情請自行搜尋如何設定picoreplayer的wifi），或是在終端機畫面設定wifi。

連上網路後，可以在瀏覽器的網址列輸入192.168.xxx.xxx：9000（你的樹莓派的IP位置及port），就可以進入LMS Player的頁面了。接著進入plugin的標籤頁，然後把waveinput這個plugin打勾。

另外，要進入Plugin頁面，要先在LMS Player(Logitech Media Server Player)的主頁面左下角點選setting，才能打開並進入Plugin頁面。 把waveinput這個plugin打勾之後，回到LMS Player(Logitech Media Server Player)的主頁面，打開左手邊的favorites，點選new favorites的按鈕後，於新增的網址列輸入wavin:default，名稱則可以自訂。

在LMS Player的主頁面右上角的下拉式選單可以點選你要轉發的裝置，例如homepod。 不過小弟之前的LMS Player一直抓不到HomePod，後來是在LMS Player的Advanced的頁面，下拉式選單選擇Airplay Bridge，將Configuration file欄位，點選Erase後再點選Generate(重新建立raopbridge.xml檔)，多重複幾次就可以抓到了。

完成後就可以把這個項目加入清單(歌單)內。然後現在可以把USB音效擷取卡插在樹梅派上，再把你的音源(ipod、黑膠唱機)接在你的音效擷取卡上。連接完成後，在LMS Player的主頁面按下播放鍵，理論上現在樹梅派已經在把它從USB音效擷取卡上擷取的音樂，轉發至HomePod上了。接下來就好好享受HomePod播放出來的音樂吧!
