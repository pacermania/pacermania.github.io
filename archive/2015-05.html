[Nexus Player]安裝Sideload app與其他app

最近由於老媽把Chromecast拿去用了，雖然說我們家還有apple tv可以使用，但是Chromecast到底還是比較威，也比較可以用一些有的沒的app，蘋果就是只能用它主機本身所提供的功能(雖然apple提供的功能確實是最大眾化的)。

原本只是想找另一支二手的Chromecast，但是後來發現Nexus Player也具有Google Cast的功能，另外還有一些內建的app可以使用，可以說是apple tv+Chromecast的合體，於是後來就買了Nexus Player了。

不過Nexus Player有一個問題，就是Nexus Player的Google Play商店和一般Android手機的商店所提供app不一樣，Nexus Player的商店只提供專門為Nexus Player設計的app（如Pandora, Netflax, Hulu Plus等），所以實際上Nexus Player並不是我們所想的一樣，要灌什麼app就灌什麼app。

所以這時候就要先用電腦，進入Google Play商店，把sideload這個app先灌進Nexus Player（可參考[教學] Nexus Player 安裝自訂 APK 程式方法 這篇文章）。

可是現在的es file explorer已經不能使用上述方法來灌進Nexus Player了，因此我們必須尋找其他的方式來安裝。後來我發現可以使用USB連線的方式，來將你想要灌的app灌進去。

首先你必須到這個網站下載adb。

接下來，就是要將你的Nexus Player用成Developer Mode。這花了我好久的時間才搞定。原來就是要在從Setting(路徑)Setting–>About–>Build Number，如下圖所示。

然後在Build Number上一直按（就像打大台電玩雷電一樣，一直連按就對了，如下圖所示。

這樣Nexus Player就可以進入Developer Mode了。

這時電腦就可以用USB轉OTG接線連接Nexus Player，並且進入Windows的終端機視窗(cmd)輸入以下指令（可參考這個網站）。

adb devices

若有連接成功，終端機視窗應該會出現以下訊息：

List of devices

xxxxxxxxxxx → 你的Nexus Player的名稱，例如 Asus Nexus Player

然後再到你儲存APK檔的目錄下，執行以下指令。

adb install 你的APK檔名（例如：adb install Google_Drive.apk或adb install ES File Explorer.apk)

把上述app sideload到你的Nexus Player裡面。

如果沒有意外的話，你的Nexus Player會多出一個Sideload的圖示。

點進去之後，就可以看到你Sideload進去的app了。

若各位有興趣，可以在露天拍賣輸入關鍵字Nexus Player，就可以找到賣家了。
